About the site

Overview

learnsanskrit.org was created in late 2009 as a set of study notes for Sanskrit students. Since then, it has continued to grow bit by bit, although it's due for a large update one of these days.

Its main feature is the Sanskrit grammar guide, which aims to help others learn Sanskrit as quickly as possible. Although this is the site's main feature, it has also been in continuous flux. The guide should grow more stable and comprehensive in the coming months.

In addition to the grammar guide, the site also features a few powerful Sanskrit tools, including Sanscript (a transliteration program) and an interface to the Monier-Williams Sanskrit-English Dictionary.

More interesting things are soon to come. But for the time being, we hope these tide you over.

Technical notes

Code

This site exists as a Flask app hosted on WebFaction.com. The site also uses Less, jQuery, and several Python packages, including SQLAlchemy.

The site was first created as a set of PHP files hosted on ibiblio.org, who generously provided free hosting. If you have a project about something of cultural significance, i.e. about something you could find in a library, and you are looking for a web host, check the ibiblio.org Collection Criteria and see if your project could find a free home.

Typesetting

All English text was set with the Dvorak Simplified Keyboard, which was created more than 70 years ago for faster and more comfortable typing. Devanagari text was originally set with Baraha Direct, part of the Baraha text-setting suite for Indic scripts. Today, most of this site's Devanagari is transliterated from Harvard-Kyoto, which is a little easier to type.
