Nouns can take different *roles* in some sentence. The simplest role just states the **subject of the sentence**: !!! examples - {{ m.x('[gajaH] pazyati', '[The elephant] sees.') }} - {{ m.x('[gajau] pazyataH', '[The two elephants] see.') }} - {{ m.x('[gajAH] pazyanti', '[The elephants] see.') }} `pazyati` implies that **only one** thing sees. `gajaH` implies that **only one** elephant performs the action. Each word implies the same number of things (one), so they are used together.

But what happens if it's followed by a `voiced` sound? !!! examples - {{ m.x('naraH dizati', 'The man points.') }} - {{ m.x('narAH dizanti', 'The men point.') }} We could try to reason about what the changes here would be; but in this one unfortunate instance, it's easiest just to memorize them: !!! examples - {{ m.x('naraH dizati -> [naro] dizati', 'The man points.') }} - {{ m.x('narAH dizanti -> [narA] dizanti', 'The men point.') }} - {{ m.x('naraH gacchati -> [naro] gacchati', 'The man goes.') }} - {{ m.x('narAH gacchanti -> [narA] gacchanti', 'The men go.') }} It's possible to explain the second change after some thought.
