To do ----- `-tum` is a root suffix that creates a new uninflected word. When added to a root meaning X, `-tum` creates a word that means "to X": !!! examples - {{ m.x('vanaM [gantuM] icchanti', 'They want [to go] to the forest.') }} A word ending in `-tum` can be used with nouns in different cases (`vanaM gantum`), just like an ordinary verb. But since these words are uninflected, they're always the same: !!! examples - {{ m.x('azvaH [gantum] icchati', 'The horse wants [to go].') }} - {{ m.x('azvau [gantum] icchataH', 'The two horses want [to go].') }} - {{ m.x('[gantum] icchAmi', 'I want [to go].') }} Using `-tum` words ------------------ `-tum` words can appear only in certain places. They're commonly used with words that imply wanting: !!! examples - {{ m.x('etAn na [hantum] icchAmi', 'I do not want [to kill] them.') }} Bhagavad Gita 1.35 or deserving: !!! examples - {{ m.x('taM santaH [zrotum arhanti]', 'The good [deserve to hear] it.') }} Raghuvamsha 1.10 - {{ m.x('na tvaM [zocitum arhasi]', 'You [should not grieve].') }} Bhagavad Gita 2.27 Making `-tum` words ------------------- ### `-i-` Just like `-ta`, `-tum` is sometimes `-tum` and sometimes `-itum`. As before, there's no simple rule for it. But the good news is that roots use it (or avoid it) consistently: !!! examples - {{ m.x('khAd -> khAdita, khAditum', 'eat -> eaten, to eat') }} - {{ m.x('car -> carita, caritum', 'walk -> walked (to), to walk') }} - {{ m.x('smR -> smRta, smartum', 'remember -> remembered, to remember') }} - {{ m.x('zuc -> zucita, zocitum', 'grieve -> grieved, to grieve') }} ### Root changes `-tum` strengthens the root vowel: !!! examples: - {{ m.x('kR -> kartum', 'do -> to do') }} There are other changes that `-tum` causes, but all the other suffixes that start with `-t` cause these changes too. We've seen enough of these suffixes now, so we'll focus on these rules in the next lesson. ### Roots in the `aya` class Roots in the `aya` class and causal roots always use `-itum`. Strangely, they use the full stem: !!! examples - {{ m.x('cur -> corayitum', 'steal -> to steal') }} - {{ m.x('dRz -> darzayitum', 'see -> to make to see') }}
