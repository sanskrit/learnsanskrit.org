Sanscript transliterator

Transliterate text from one script to another. Sanscript supports all modern Indian scripts and the most popular romanizations. It's fast, robust, and thoroughly tested. Although Sanscript focuses on Sanskrit, it has strong support for other Indian languages.

Monier-Williams dictionary

A cleaner version of the Monier-Williams Sanskrit-English Dictionary, the largest and most commonly used Sanskrit-English dictionary. This version contains some nice improvements:

  • Look up multiple words at once
  • Enter input in nearly any common format
  • View output in any Indian script
  • Color-coded information for faster reading

Dictionary data comes from the Cologne Digital Sanskrit Lexicon project.
