Just as simple tin and copper can combine to make bronze, two vowels can combine to make a compound vowel.

Compound vowels vowels are a crucial part of Sanskrit and are used in simple but powerful ways. But for now, let us just pronounce them.

The vowels

Sanskrit has four compound vowels. Each is a long vowel. And each is made by a different combination.

Since the compound vowels are combinations of two vowels, they are similar to nothing.

{{ 'a/A'|i }} + simple vowel

Consider the combinations {{ 'a'|i }} + {{ 'i'|i }} and {{ 'a'|i }} + {{ 'u'|i }}. These combine in an obvious way:

    {{ ex('a + i -> ai', dev=false, iast=true) }} {{ ex('a + u -> au', dev=false, iast=true) }}

But it can be tiresome to keep these two sounds separate. So, the early Sanskrit speakers blended the two sounds into something a little easier:

    {{ sound('e', '"a" in "mane"') }} {{ sound('o', '"o" in "go"') }}

The other combinations ({{ 'aI, Ai, AI, aU, Au, AU'|i }}) blend in the same way.

As you pronounce {{ 'e'|i }} and {{ 'o'|i }}, try to make the sound "flat" and constant. If you are a native English speaker, this can be hard; English "e" sounds like Sanskrit {{ 'ei'|i }} and English "o" sounds like Sanskrit {{ 'ou'|i }}.

For now, let's ignore combinations with {{ 'R'|i }} and {{ 'RR'|i }}.

{{ 'a/A'|i }} + compound vowel

Consider the combinations {{ 'a + e'|i }} and {{ 'a + o'|i }}. These combinations are not obvious. But if we remember that {{ 'e'|i }} comes from {{ 'a + i'|i }} and that {{ 'o'|i }} comes from {{ 'a + u'|i }}, they become easy:

    {{ ex('a + ai -> Ai', dev=false, iast=true) }} {{ ex('a + au -> Au', dev=false, iast=true) }}

But it can be tiresome to spend so much time pronouncing a vowel. So, the early Sanskrit speakers made the {{ 'A'|i }} sound a little shorter:

    {{ sound('ai', '"i" in "fight"') }} {{ sound('au', '"ow" in "cow"') }}

The other combinations ({{ 'Ae, Ao'|i }}) blend in the same way.

Can we combine {{ 'ai'|i }} and {{ 'au'|i }} with anything? We can try:

    {{ ex('a + ai -> Ai', dev=false, iast=true) }} {{ ex('a + au -> Au', dev=false, iast=true) }}

But they shorten back to {{ 'ai'|i }} and {{ 'au'|i }}, with no changes. The other combinations ({{ 'A+ai, A+au'|i }}) do the same.

As you pronounce {{ 'ai'|i }} and {{ 'au'|i }}, try to make the "a" part of {{ 'ai'|i }} and {{ 'au'|i }} sound just like the vowel {{ 'a'|i }}. The shorter it is, the better.

Blending compound vowels

When two vowels are similar, they blend easily:

    {{ ex('bAlAya Aha -> bAlAyAha', 'He spoke for the boy.', iast=true) }} {{ ex('gacchati IzvaraH -> gacchatIzvaraH', 'The lord goes.', iast=true) }}

And if they are not similar, they still blend easily. {{ 'a'|i }} and {{ 'A'|i }} combine like they do above:

    {{ ex('sA icchati -> secchati', 'She wants.', iast=true) }} {{ ex('tasya udakam -> tasyodakam', 'his water', iast=true) }} {{ ex('bAlasya odanam -> bAlasyaudanam', "the boy's rice", iast=true)}} {{ ex('tasya aizvaryam -> tasyaizvaryam', "his power", iast=true)}}

In the last example, note that {{ 'a + ai'|i }} combine with no change. Also, remember that nothing is similar to a compound vowel.

In the wild

Try to blend the words in the phrases below:

    {{ ex('pazya etAm', iast=true) }} {{ ex('hatvA etAn', iast=true) }} {{ ex('sakhA iti matvA', iast=true) }} {{ ex('ca oSadhIH', iast=true) }} {{ ex('na asad AsIn na u sad AsIt', iast=true) }}

These words blend as you would expect:

    {{ ex('pazyaitAm', 'Look at this (army of the Pandavas, O master).', cite='Bhagavad Gita 1.3', iast=true) }} {{ ex('hatvaitAn', 'Having killed them, …', cite='Bhagavad Gita 1.36', iast=true) }} {{ ex('sakheti matvA', 'Thinking (of you) as a friend, …', cite='Bhagavad Gita 11.41', iast=true) }} {{ ex('cauSadhIH', 'And (I nourish all) the plants.', iast=true, cite='Bhagavad Gita 15.13') }} {{ ex('nAsad AsIn no sad AsIt', 'Then there was neither nothing nor anything.', cite='Nāsadīya Sūkta', iast=true) }}
