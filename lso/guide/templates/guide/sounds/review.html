Blending

Sometimes, it can be tiresome to pronounce certain sounds next to each other. The earliest Sanskrit speakers solved this problem by blending words together.

Blending vowels

It is easy to blend vowels. Simple vowels are the easiest:

    {{ ex('bAlAya Aha -> bAlAyAha', 'He spoke for the boy.', iast=true) }} {{ ex('sA Apnoti -> sApnoti', 'She obtains.', iast=true) }} {{ ex('gacchati IzvaraH -> gacchatIzvaraH', 'The lord goes.', iast=true) }} {{ ex('gacchati madhu udakam -> gacchati madhUdakam', 'He goes to the sweet water.', iast=true) }}

Otherwise, vowels can blend in several ways. They can combine:

    {{ ex('sA icchati -> secchati', 'She wants.', iast=true) }} {{ ex('tasya udakam -> tasyodakam', 'his water', iast=true) }} {{ ex('bAlasya odanam -> bAlasyaudanam', "the boy's rice", iast=true)}} {{ ex('tasya aizvaryam -> tasyaizvaryam', "his power", iast=true)}}

or one can become a semivowel:

    {{ ex('gacchati azvaH -> gacchaty azvaH', 'The horse goes.', iast=true) }} {{ ex('sAdhu Aste -> sAdhv Aste', 'He sits well.', iast=true) }} {{ ex('tejasvi nAv adhItam astu', 'May our studies be glorious.', cite='Upanishads (various)', iast=true) }}

Blending consonants

It is easy to blend consonants, too. Stops can become nasals:

    {{ ex('tat na Asyam -> tan nAsyam', 'That is not a mouth.', iast=true) }} {{ ex('rAT naraH -> rAN naraH', 'The king is a man.', iast=true) }}

or they can just become voiced:

    {{ ex('vAk eva -> vAg eva', 'speech itself', iast=true) }} {{ ex('tat yacchati -> tad yacchati', 'He restrains it.', iast=true) }} {{ ex('sa rAT bhavati -> sa rAD bhavati', 'He becomes a king.', iast=true) }} {{ ex('na asat AsIt na u sat AsIt tadAnIm -> nAsad AsIn no sad AsIt tadAnIm', 'Then there was neither nothing nor anything.', cite='Nāsadīya Sūkta', iast=true) }}

{{ 't'|i }}, especially, blends very easily:

    {{ ex('tat cApam -> tac cApam', 'That is a bow.', iast=true) }} {{ ex('tat TIkAsu -> taT TIkAsu', 'That is in the commentaries.', iast=true) }} {{ ex('tat jAyate -> taj jAyate', 'That is born.', iast=true) }} {{ ex('tat Damarau -> taD Damarau', 'That is in the drum.', iast=true) }}

Blending the {{ 'visarga'|i }}

Just like {{ 't'|i }}, the {{ 'visarga'|i }} blends often and easily:

    {{ ex('naraH carati -> naraz carati', 'The man walks.', iast=true) }} {{ ex('naraH tarati -> naras tarati', 'The man crosses.', iast=true) }} {{ ex('tAH TIkAH -> tAS TIkAH', 'Those are commentaries.', iast=true) }}
