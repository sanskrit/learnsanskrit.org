The two sounds here are fundamentally different from the others we have studied. They appear only because of blending. They can be pronounced in multiple ways. They must follow vowels. And although they seem to be consonants, the tradition calls them something else. Each sound has its own special term.

{{ 'anusvAra'|i }}

    {{ sound('aM', '(no match)', 'anusvara') }}

This sound is called the {{ 'anusvAra'|i('dfn') }} ("after-sound"). It is a "pure nasal" sound that appears only in front of consonants.

It is difficult to pronounce a "pure nasal." But the {{ 'anusvAra'|i }} is easy to pronounce. Generally, it uses the same point of sound as the sound that follows it:

{{ 'visarga'|i }}

    {{ sound('aH', '(no match)', 'visarga') }}

This sound is called the {{ 'visarga'|i('dfn') }} ("release").

Originally, the {{ 'visarga'|i }} was probably just like the "h" in "house." We could think of it as an "s" sound pronounced at the soft palate. But today, it is usually pronounced as an echo of the vowel before it: {{ 'aH'|i }} like {{ 'aha'|i }}, and {{ 'iH'|i }} like {{ 'ihi'|i }}.

Blending the {{ 'visarga'|i }}

The {{ 'visarga'|i }} is a difficult sound. So wherever possible, it blends with the letters around it. In front of unvoiced consonants, the {{ 'visarga'|i }} becomes the "s" sound with the same point of sound:

    {{ ex('naraH carati -> naraz carati', 'The man walks.', iast=true) }} {{ ex('naraH tarati -> naras tarati', 'The man crosses.', iast=true) }} {{ ex('tAH TIkAH -> tAS TIkAH', 'Those are commentaries.', iast=true) }}

This change also occurs in front of other "s" sounds, like {{ 'z'|i }} and {{ 's'|i }}. But surprisingly, the change is rarely written out:

    {{ ex('naraH zocati (naraz zocati)', 'The man grieves.', iast=true) }} {{ ex('naraH smarati (naras smarati)', 'The man remembers.', iast=true) }}
