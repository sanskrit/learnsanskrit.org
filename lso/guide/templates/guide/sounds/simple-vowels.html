Sounds and letters

When most people think of written Sanskrit, they think of Devanagari:

    {{ ex('tejasvi nAvadhItamastu', 'May our studies be glorious.', cite='Upanishads (various)', iast=false) }}

But although Devanagari is standard now, it wasn't always. Historically, every Indian script has been used to write Sanskrit:

  • {{ 'tejasvi nAvadhItamastu'|d(to='bengali') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='gujarati') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='gurmukhi') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='kannada') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='malayalam') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='oriya') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='tamil') }}
  • {{ 'tejasvi nAvadhItamastu'|d(to='telugu') }}

This fact is deeply connected to the Sanskrit tradition, which has always valued speech over writing. Even when writing was abundant and widely known, the Vedas and other important texts were learned from the mouth of a teacher and memorized so that they could be taught later on. And although it is weaker now, this tradition has survived to the present day. This emphasis on speech over writing helps to explain why words are blended in so many Sanskrit texts.

But it also leads to a more practical matter. If Sanskrit has no script of its own, we can choose whatever script we like.

Devanagari is an obvious choice. But Devanagari takes some time to learn, especially if you have never learned another Indian script. Moreover, Devanagari was not built for Sanskrit, and it can be awkward and clumsy when used to write it.

Instead, we could use romanized Sanskrit. It is almost as common as Devanagari, and it was built to be easy to learn:

  • {{ 'tejasvi nAvadhItamastu'|i }}

As a compromise, this guide will use romanized Sanskrit and switch to Devanagari over time. With this approach, we can spend less time on reading and writing and more time on Sanskrit.

And speaking of Sanskrit, let us continue with the alphabet.

Seven vowels

Four of these vowels have English counterparts:

    {{ sound('i', '"i" in "bit"') }} {{ sound('I', '"ee" in "teeth"', 'ii') }} {{ sound('u', '"u" in "put"') }} {{ sound('U', '"oo" in "mood"', 'uu') }}

Three do not:

    {{ sound('R', '(no match)', 'r') }} {{ sound('RR', '(no match)', 'rr') }} {{ sound('lR', '(no match)', none) }}

{{ 'lR'|i }} is extremely rare. Most texts do not have it, and it does not have a long form. Generally, you can pronounce it however you like.

Short and long

We have studied 9 vowels so far. Of these, five are short:

  • {{ 'a'|i }}
  • {{ 'i'|i }}
  • {{ 'u'|i }}
  • {{ 'R'|i }}
  • {{ 'lR'|i }}

and four are long:

  • {{ 'A'|i }}
  • {{ 'I'|i }}
  • {{ 'U'|i }}
  • {{ 'RR'|i }}

Together, these nine vowels are called simple vowels.

Blending simple vowels

The simple vowels are easy to blend:

    {{ ex('bAlAya Aha -> bAlAyAha', 'He spoke for the boy.', iast=true) }} {{ ex('sA Apnoti -> sApnoti', 'She obtains.', iast=true) }} {{ ex('gacchati IzvaraH -> gacchatIzvaraH', 'The lord goes.', iast=true) }} {{ ex('gacchati madhu udakam -> gacchati madhUdakam', 'He goes to the sweet water.', iast=true) }}

In these sentences, the vowels that blend resemble each other. {{ 'A'|i }} blends with {{ 'A'|i }}, {{ 'i'|i }} blends with {{ 'I'|i }}, {{ 'u'|i }} blends with {{ 'u'|i }}, and so on. In each case, the vowels are roughly the same, although they might have different lengths.

Let's call such vowels similar. For example, {{ 'R'|i }} is similar to {{ 'R'|i }} and {{ 'RR'|i }}, but it is not similar to {{ 'u'|i }}.

In the wild

Try to blend the words in the phrases below:

    {{ ex('na anyadasti iti', iast=true) }} {{ ex('eSa tu uddezataH', iast=true) }}

These phrases are from the Bhagavad Gita:

    {{ ex('nAnyadastIti', '(The unwise, who delight in the letter of the Vedas and proclaim) "there is nothing else", …', cite='Bhagavad Gita 2.42', iast=true) }} {{ ex('eSa tUddezataH', '(What I have declared) is just an example (of my many splendors.)', cite='Bhagavad Gita 10.40', iast=true) }}

These two blends are common all throughout Sanskrit literature.
