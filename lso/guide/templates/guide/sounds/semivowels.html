How do two vowels blend together? If they are similar, they become long:

    {{ iex('bAlAya Aha -> bAlAyAha', 'He spoke for the boy.') }}

If they are not similar, the blend depends on the first vowel. {{ 'a'|i }} and {{ 'A'|i }}, for example, combine to create compound vowels:

    {{ iex('sA icchati -> secchati', 'She wants.') }}

But there are other combinations that are more puzzling:

    {{ iex('gacchati azvaH', 'The horse goes.') }} {{ iex('sAdhu Aste', 'He sits well.') }}

To blend the vowels in these sentences, we need a new kind of letter.

Semivowels

Consider the combinations {{ 'i'|i }} + {{ 'a'|i }} and {{ 'u'|i }} + {{ 'a'|i }}. These combine in an obvious way:

    {{ iex('i + a -> ia') }} {{ iex('u + a -> ua') }}

It can be tiresome to keep these two sounds separate. But these sounds do not blend easily. They fight for space, like two wrestlers in the ring. And only one of them can remain.

Instead of blending, one of the sounds collapses and becomes shorter. Wherever possible, the first sound is the one that shortens:

    {{ iex('i + a -> ya') }} {{ iex('u + a -> va') }}

These shortened vowels are called semivowels. And apart from {{ 'a'|i }} and {{ 'A'|i }}, every vowel has one:

    {{ sound('ya', '"y" in "yellow"') }} {{ sound('ra', '(no match)') }} {{ sound('la', '"l" in "loose"') }} {{ sound('va', '"v" in "vase"') }}

Since semivowels can only exist around other vowels, they are all listed with the vowel {{ 'a'|i }}. As you pronounce these letters, keep them as short as possible.

Fake compound vowels

In the previous lesson, we saw that we create compound vowels by combining two simple vowels. And we studied four compound vowels: {{ 'e'|i }}, {{ 'o'|i }}, {{ 'ai'|i }}, and {{ 'au'|i }}. But none of these compound vowels use {{ 'R'|i }}, {{ 'RR'|i }}, or {{ 'lR'|i }}. Can we use these three vowels at all? Yes and no. Using semivowels, we can create "fake" compound vowels:

    {{ iex('a + R -> ar') }} {{ iex('a + RR -> ar') }} {{ iex('a + lR -> al') }} {{ iex('A + lR -> al') }}

And just like the "real" compound vowels, these "fake" compound vowels can combine again:

    {{ iex('a + ar -> Ar') }} {{ iex('A + ar -> Ar') }} {{ iex('a + al -> Al') }} {{ iex('A + al -> Al') }}

Blending vowels

When vowels cannot blend or combine, one of them becomes a semivowel:

    {{ iex('gacchati azvaH -> gacchaty azvaH', 'The horse goes.') }} {{ iex('sAdhu Aste -> sAdhv Aste', 'He sits well.') }}

With compound vowels

This applies to compound vowels, too. We just have to remember where they come from. For example, {{ 'au'|i }} comes from {{ 'a + a + u'|i }}, or {{ 'Au'|i }}. So, we get:

    {{ iex('azvau icchati -> azvAu icchati -> azvAv icchati', 'He wants the two horses.') }}

In the wild

Try to blend the words in the phrases below:

    {{ iex('bhavati iti anuzuzruma') }} {{ iex('na tu eva ahaM jAtu na Asam') }} {{ iex('tejasvi nau adhItam astu') }} {{ iex('yadi api ete na pazyanti') }}

These words blend as you would expect:

    {{ iex('bhavatIty anuzuzruma', '(They dwell eternally in hell) — so we have heard.', cite='Bhagavad Gita 1.44') }} {{ iex('na tv evAhaM jAtu nAsam', 'Never was I ever not.', cite='Bhagavad Gita 2.12') }} {{ iex('tejasvi nAv adhItam astu', 'May our studies be glorious.', cite='Upanishads (various)') }} {{ iex('yady apy ete na pazyanti', 'But even if they do not see, …', cite='Bhagavad Gita 1.38') }}

Other sounds

We can now describe how the Sanskrit vowels blend and interact. This knowledge is highly useful and will be especially important later on.

But there are still many other sounds to consider. Let's take a break from the vowels and see what some of these sounds are. These sounds are much simpler, and they will be much easier to learn.
