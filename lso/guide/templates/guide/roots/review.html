And that is all! Starting with a basic background in Sanskrit grammar, we built up some of Sanskrit's most powerful devices. We learned that all verbs come from a set of basic roots, which are sorted into various classes. We learned that nouns have a gender and that this gender does not always make sense. Inserted {{ 'i'|i }}

In front of certain suffixes, a root might use a special inserted {{ 'i'|i }} vowel. When used with these suffixes, roots will consistently follow one of three patterns:

  • They always use {{ 'i'|i }}.
  • They never use {{ 'i'|i }}.
  • They sometimes use {{ 'i'|i }}.

So far, we have seen three suffixes that cause this behavior: {{ '-sya'|i }}, {{ '-ta'|i }}, and {{ '-tvA'|i }}.
