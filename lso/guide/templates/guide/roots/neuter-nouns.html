Although this unit focuses on roots and verbs, we should also spend some time on nouns and adjectives. After all, our pool of nouns is still quite small. And there are only so many sentences we can write about a {{ 'nara'|i }} in a {{ 'grAma'|i }} with an {{ 'azva'|i }} before we start getting bored.

For now, let's learn three new nouns:

    {{ ex('phalam icchanti gajAH', 'The elephants want the fruit.') }} {{ hex('phalair vinA narA grAmaM gacchanti', 'The men go to the village without the fruits.') }} {{ ex('vane zRNomi gajAn', 'I hear elephants in the forest.') }} {{ hex('siMho kRSNagajair vaneSu tiSThati ', 'The lion stands with the black elephants among the forests.') }} {{ ex('nagarAd gacchanti sundarA narAH', 'Handsome men come from the city.') }} {{ hex('gRhe narAH phalam corayanti', 'The men steal the fruit in the house.') }}

These nouns behave just like the ones we have already seen. But let us test these words a little more:

    {{ ex('narasya putro bhavati', 'The man has a son. (Literally, "Of the man, there is a son.")') }} {{ ex('devasya phalam asti', 'The god has a fruit.') }} {{ ex('nagare phalaM tiSThati', 'A fruit stands in the city.') }} {{ ex('azvau pazyAmi', 'I see two horses.') }} {{ ex('phale labhadhve', 'You all obtain the two fruits.') }} {{ ex('phale icchataH', 'The two of them want two fruits.') }} {{ ex('zvetagajAn krINAsi', 'You buy white elephants.') }} {{ ex('phalAni labhAvahe vane', 'The two of us obtain fruits in the forest.') }} {{ ex('phalAni khAdAmaH', 'We eat the fruits.') }}

Now our sentences are more interesting. Even though {{ 'phala'|i }} and {{ 'nara'|i }} both end in {{ 'a'|i }}, these nouns seem to use different endings. Why does this happen?

Gender

Nouns have two main properties. The first is the noun case, which defines the role that the noun plays in some action. The second is the number of things that the noun refers to. We can use these two properties to describe all of the nouns that we have seen so far:

    {{ ex('gajena', 'with the elephant (case 3, refers to one thing)') }} {{ ex('narayoH', 'of the two men (case 6, refers to two things)') }} {{ ex('grAmebhyaH', 'from the villages (case 5, refers to many thing)') }}

But nouns have a third property, which is called gender. In one sense, Sanskrit gender is like the genders we have in real life. So we have masculine nouns for male things:

    {{ ex("bAlo 'zvaM corayati", 'The boy steals the horse.') }} {{ ex('gajaH pazyati', 'The (male) elephant sees.') }}

feminine nouns for female things:

    {{ ex('bAlA pacati', 'The girl cooks.') }} {{ ex('gajA pazyati', 'The (female) elephant sees.') }}

and neuter nouns for everything else:

    {{ ex('grAme zvetaM phalaM bhavati', 'There is a white fruit in the village.') }} {{ ex('gacchasi nagare sundare', 'You go to the two cities.') }} {{ ex('kRSNeSu vaneSu carAmaH', 'We walk in the black forests.') }}

These examples tell us a lot about Sanskrit nouns:

  • A noun has a basic stem, like {{ 'gaja'|i }} ("elephant") or {{ 'bAla'|i }} ("child").
  • A noun's endings define the gender, case, and number of the full noun.
  • Adjectives must have the same gender, case, and number as the words they describe.
  • The "grammatical gender" of the noun matches the real-life gender of the real object. (It makes intuitive sense that a male elephant would be described with a masculine noun.)

But unfortunately, this is not the whole story. In truth, gender does not always make sense. Why is {{ 'grAma'|i }} male when {{ 'nagara'|i }} and {{ 'vana'|i }} are neuter? There is no good answer. And it is often the case that words have certain genders solely because of how they were created. Words that end in {{ 'tra'|i }}, for example, are usually neuter. And words that end in {{ 'ana'|i }} are usually neuter too.

Also, a noun stem cannot always use endings in any gender. We saw above that {{ 'gaja'|i }} could be both masculine and feminine. This makes biological sense. But {{ 'grAma'|i }}, for example, is nearly always a masculine noun, and it never uses feminine endings.

This means that gender is not intuitive. When we learn a new word like {{ 'nagara'|i }}, we can only understand it if we know what genders it uses. And these genders do not always correspond to something that makes sense in the real world.

Back to neuter nouns

The examples above show how much the gender of a noun can change the endings it uses. Fortunately, the masculine endings and neuter endings are almost identical. They differ in case 1, case 2, and case 8:

{{ nominal('Pala', 'n', cases='128') }}

These three cases use almost exactly the same endings. Most neuter nouns have this property. All other cases are identical to the cases for male nouns like {{ 'gaja'|i }}, {{ 'nara'|i }}, and {{ 'grAma'|i }}:

{{ nominal('Pala', 'n', cases='34567') }}

Ambiguities

Since neuter endings reuse many of the same endings, they might seem especially confusing. But this is almost never a problem. Consider that neuter nouns tend to refer to inanimate things, like objects:

    {{ hex('phalaM labhase', 'You obtain the fruit.') }}

places:

    {{ ex('vanaM dahanti', 'They burn the forest.') }}

and concepts:

    {{ ex('nimittaM pazyati', 'He sees an omen.') }}

So we can rule out case 8, which usually sounds ridiculous:

    {{ ex('sundarANi kadalIphalAni gacchAmi', '(?) O beautiful bananas, I am going.') }}

And with most of the verbs we have seen so far, case 1 is equally ridiculous:

    {{ ex('vanaM pazyati', '(?) The forest is watching.') }} {{ ex('nagaraM carati', '(?) The city walks.') }}

Instead, neuter nouns are usually used in cases 2 through 7:

    {{ hex('vanaM pazyati', 'He sees the forest.') }} {{ hex('nagare phalaM khAdasi', 'You eat the fruit in the city.') }} {{ hex('devAnAM vanAni rakSati', 'He protects the forests of the gods.') }} {{ hex('sundarAn nagarAc carati', 'He walks from the beautiful city.') }} {{ hex('zvetAni phalAnIcchanti', 'They want the white fruits.') }}

Non-ambiguity

Usually, vowels blend together:

    {{ ex('nareNAzvaM krINAmi', 'I buy the horse with the man.') }} {{ ex('phalAnIcchAmi', 'I want the fruits.') }} {{ ex('grAma icchasi', 'You want in the village.') }}

But some vowels are exempt from sandhi changes. That is, they cause no sandhi changes and are unchanged by sandhi.

When {{ 'e'|i }} ends a word that refers to exactly two things, it is exempt:

    {{ hex('nagara icchanti', 'They want in the two cities.') }} {{ hex('nagare icchanti', 'They want the two cities.') }}

This applies to verbs, too:

    {{ hex("labhe 'zvam", 'I obtain the horse.') }} {{ hex('labhAvahe azvam', 'The two of us obtain the horse.') }} {{ hex("labhate 'zvam", 'He obtains the horse.') }} {{ hex('labhete azvam', 'The two of them obtain the horse.') }}

In the wild

Our goal is to read Sanskrit texts. So let's look at an example from the Bhagavad Gita. Here, the word {{ 'kezava'|i }} is a name of Krishna:

    {{ ex('nimittAni ca pazyAmi viparItAni kezava', 'And I see terrible omens, Krishna.', cite='Bhagavad Gita 1.31') }}

Now that we have a good foundation in Sanskrit, every new lesson will unlock dozens of new words and phrases. And by the end of the unit, we will be able to read and understand many of the words in the Bhagavad Gita. In fact, we will study the word {{ 'viparIta'|i }} in detail in just a few lessons; by the end of the unit, you will be able to fully understand this line.
