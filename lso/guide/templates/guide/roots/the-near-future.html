Sanskrit roots can grow into dozens of different stems. So far, we have focused on only one stem: the one used in the present tense. This "present stem" falls into ten different classes, and we have studied seven of these classes so far. But despite this complexity, our picture of the Sanskrit verb is still quite basic:

So let's start making this picture more interesting:

Now we have a new stem to consider. It's similar to the ones we've seen already. And it even uses the same endings. But it means something quite different:

    {{ ex('narasya putro vIro bhaviSyati', "The man's son will be a hero.") }}

These verbs let us describe what will happen. Such verbs are called future tense verbs. But since Sanskrit has several verbs that describe the future tense, let's say that these verbs describe the near future. That is, they describe what will happen soon.

The near future appears almost 60 times in the Bhagavad Gita. Some of these verbs are below. These examples aren't very exciting, but they do show some of the variety that these forms have:

    {{ ex('jJAsyasi', 'You will know', cite='BG 7.1') }} {{ ex('drakSyasi', 'You will see', cite='Bhagavad Gita 4.35') }} {{ ex('kariSye', 'I will do', cite='BG 18.61') }}

Creating future stems

It is easy to create the future stem:

  1. Strengthen the root.
  2. Add the suffix {{ 'sya'|i }}.

The future stem uses the same endings as the present tense. We use {{ 'parasmaipada'|i }} or {{ 'Atmanepada'|i }} endings depending on what the verb uses in the present tense. For example, {{ 'jJA'|i }} uses both {{ 'parasmaipada'|i }} ({{ 'jAnAti'|i }}) and {{ 'Atmanepada'|i }} ({{ 'jAnIte'|i }}), so it uses both in the near future as well:

    {{ ex('jJA -> jJAsyati', 'know -> He will know.') }} {{ ex('jJA -> jJAsyate', 'know -> He will know.') }}

But this description is incomplete. Several stems seem to do something else instead:

    {{ ex('corayiSyAmi', 'I will steal') }} {{ ex('drakSyasi', 'You will see') }} {{ ex('kariSye', 'I will do') }}

Three different issues are involved here:

Roots in the {{ 'aya'|i }} class

Roots in the {{ 'aya'|i }} class are strange. Even though all other roots form future stems without using information from their verb classes, roots in the {{ 'aya'|i }} class use this information anyway:

    {{ ex('cintayati', 'He thinks.') }} {{ ex('cintayiSyanti', 'They will think.') }}

Sandhi

{{ 'drakSya'|i }} is the future stem of {{ 'dRz'|i }}. {{ 'dRz'|i }} irregularly strengthens to {{ 'draz'|i }} instead of *{{ 'darz'|i }}. But even if that's the case, we would expect that the future stem would be {{ 'drazsya'|i }}. Why isn't this the case?

The answer is that {{ 'drazsya'|i }} is just difficult to pronounce. The shift from {{ 'z'|i }} to {{ 's'|i }} is tricky and unclear, especially when speaking quickly. The earliest Sanskrit speakers felt the same way, and they solved this problem by combining {{ 'z'|i }} and {{ 's'|i }} into something simpler:

Within a word, {{ 'z'|i }} and {{ 's'|i }} combine to {{ 'kS'|i }}.

{{ 'kS'|i }} is clearer than {{ 'zs'|i }} and does not require such a nimble tongue. The change from {{ 's'|i }} to {{ 'S'|i }} is common throughout Sanskrit, and we have seen examples of it already.

Inserted {{ 'i'|i }}

A significant number of future stems insert an {{ 'i'|i }} vowel between the root and {{ 'sya'|i }}. {{ 'isya'|i }} becomes {{ 'iSya'|i }} due to sandhi:

    {{ ex('car -> cariSyati', 'He will walk.') }}

{{ 'na'|i }}, {{ 'ca'|i }}, and {{ 'eva'|i }}

As our grasp of Sanskrit becomes stronger, we will be able to understand more and more of the language. But some concepts are simple enough that we don't need to spend full lessons on them. Here, let's look at three simple uninflected words:

{{ 'na'|i }}

{{ 'na'|i }} is related to the English word "no," and it means nearly the same thing:

    {{ ex('na bhaviSyanti', 'They will not be', cite='BG 11.32') }}

{{ 'ca'|i }}

{{ 'ca'|i }} means "and." It can connect nouns to nouns:

    {{ ex('naro vIro gajaz ca siMhaM zRNvanti', 'The man, the hero, and the elephant hear the lion.') }}

and sentences to sentences:

    {{ ex('azvA gacchanti gamiSyanti ca', 'The horses go and will go.') }}

Unlike the English "and," {{ 'ca'|i }} appears at the very end of the list of connected words. This means that {{ 'ca'|i }} can never start a sentence.

{{ 'eva'|i }}

In English, we emphasize words by speaking them differently:

    {{ ex(en="The man will steal the hero's black chariot.") }}

In Sanskrit, we emphasize words by adding {{ 'eva'|i }} after them:

    {{ ex('nara eva kRSNaM vIrarathaM corayiSyati', "The man will steal the hero's black chariot.") }}

{{ 'eva'|i }} is sometimes translated as "indeed" or "truly."

In Sanskrit poetry, {{ 'eva'|i }} is often used to fill gaps in a verse's meter. When used in this way, it does not have any strong meaning:

    {{ ex('mAmakAH pANDavAzcaiva', 'My (sons) and the Pandavas', cite='Bhagavad Gita 1.1') }}

{{ 'na'|i }}, {{ 'ca'|i }}, and {{ 'eva'|i }}

Using these three words, we can read a full line from the Bhagavad Gita:

    {{ ex('na caiva na bhaviSyAmaH', 'And truly, we will always be. (Literally, "we will not not be.")', cite="Bhagavad Gita 2.12") }}
