In Vedic Sanskrit, verbs are often used with some special uninflected words. In the example below, you can see one of these special words used with {{ 'car'|i }}. Although {{ 'car'|i }} normally means "walk," the uninflected word {{ 'A'|i }} modifies the basic meaning of the verb:

    {{ ex('kam A janaM carati kAsu vikSu', 'What people does he visit, and in what dwellings?', cite='Rigveda 6.21.3') }}

In later Sanskrit, such words are always stuck right at the beginning of the verb:

    {{ ex('A + car -> Acar -> Acarati', 'near + walk -> walk near; visit -> he visits') }}

The Ashtadhyayi has a special name for these uninflected words. But let's be general and just called them verb prefixes. Just as "go out" and "go with" mean something different from just "go," a root used with a verb prefix usually means something different from a verb used without one.

What prefixes mean

By using prefixes, we can use the basic set of verbs to create thousands more, each with a precise meaning:

    {{ ex('vi + jJA -> vijJA', 'out/apart + know -> perceive, discern, discriminate, recognize') }} {{ ex('ava + tRR -> avatRR', 'downward + cross -> descend, appear (') }}

And we can use prefixed roots just as we do any other root. For example, we can use root suffixes to create new words:

    {{ ex('vijJAnaM yajJaM tanute', "{{ 'vijJAna'|i }} (insight, discriminating knowledge) performs the sacrifice."|render, cite='Taittiriya Upanishad 2.5') }} {{ ex('vi + sRj -> visRj -> visarga', "out/apart + emit -> emit outward, eject -> that which is ejected; a {{ 'visarga'|i }}"|render) }} {{ ex('avatRR -> avatAra', 'descend, appear -> descent, appearance, manifestation; avatar') }}

But a verb prefix does not always cause such a clear change in meaning. Like all phrases used by human beings, roots combined with prefixes tend to have more specialized meanings that are difficult to predict:

    {{ ex('ava + gam -> avagam -> avagacchati', 'downward + go -> He understands.') }}

And in some verbs, prefixes do almost nothing. In the example below, the change is essentially non-existent. Some say that the prefix makes the sense of the verb more intense:

    {{ ex('sam + jan -> saMjan -> saMjAyate', 'with/together + be born -> be born -> He is born.') }}

Still, prefixes do usually cause a noticeable change. And if we use multiple prefixes, we can stack these changes on top of each other:

    {{ ex('upa + sam + gam -> upasaMgacchantyazvaM mRgA', 'near/toward + with + go -> The deer approach the horse together.') }}

Sanskrit has about twenty different verb prefixes, each with its own basic meanings. For now, let's restrict ourselves to the five featured above:

{{ 'A'|sa1 }}
near
{{ 'ava'|d }}
downward
{{ 'upa'|d }}
near, toward, up to, next to
{{ 'vi'|d }}
outward, apart
{{ 'sam'|d }}
with, together

Using prefixes

Prefixes combine using normal sandhi rules. The one exception is that if the prefix ends in {{ 'a'|i }} and the root starts with {{ 'R'|i }}, then the two combine to {{ 'Ar'|i }}:

    {{ ex('upa + R -> upArcchati', 'near/toward + go -> He goes near.') }}

In the wild

Except for {{ 'ava'|i }}, all of the prefixes above are very common. They are so common, in fact, that all of them appear in verse 2 of the Bhagavad Gita:

    {{ ex('dRSTvA tu pANDavAnIkaM viUDhaM duryodhanas tadA', 'Having seen the army of the Pandavas arrayed, Duryodhana then') }} {{ ex('AcAryam upasaMgamya rAjA vacanam abravIt', 'approached his master and spoke this statement.') }}

This verse contains a few unfamiliar concepts. But we can understand a surprising amount of it. {{ 'pANDavAnIka'|i }} is a {{ 'tatpuruSa'|i }} compound:

    {{ ex('pANDavAnAm anIkam -> pAndavAnIkam', 'The army of the Pandavas') }}

Two words are formed using the {{ '-ana'|i }} suffix:

    {{ ex('vac -> vacana', 'speak -> an act of speaking; speech, word, statement') }} {{ ex('yudh -> yodhana', 'fight -> an act of fighting; war') }}

And {{ 'AcArya'|i }} is a word derived from {{ 'AcAra'|i }}:

    {{ ex('A + car -> Acar -> AcAra', 'near + walk -> walk toward; visit; practice, undertake -> undertakings, practice, custom, precepts') }}

"Master" is an inaccurate translation for {{ 'AcArya'|i }}. More accurately, an {{ 'AcArya'|i }} is one who knows the {{ 'AcAra'|i }}, and it is often attached to names as a sign of respect. Grammatically, this is a case 1 {{ 'tatpuruSa'|i }}. The {{ 'vigraha'|i }} is on the left:

    {{ ex('rAmAnuja eva AcAryaH -> rAmAnujAcArya', 'Ramanuja himself is an acharya -> Ramanujacharya') }}
We will study how to create the word {{ 'AcArya'|i }} in a future lesson. And speaking of future lessons, the next two lessons will explain three more words from this verse: {{ 'dRSTvA'|i }}, {{ 'vyUDham'|i }}, and {{ 'upasaMgamya'|i }}.
