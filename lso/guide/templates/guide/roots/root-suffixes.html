Sanskrit is a deep and rewarding language. But nothing is deeper, more rewarding, or more fascinating than the ability to create new words. By applying simple rules to verb roots, we can quickly create words of great subtlety and power.

In a sense, all of Sanskrit reduces to making complex words from simple parts. And in fact, most of the rules of the Ashtadhyayi are concerned with creating words of one kind or another.

But in this lesson, we will focus on a specific part of word creation: creating nouns and adjectives from verb roots. We do so by using a suffix, a simple group of sounds that is attached to the end of something:

    {{ ex('gam -> gamana', 'go -> the act of going') }}

A suffix that attaches directly to a verb root is a root suffix. Sanskrit has dozens of different root suffixes, and we will study many of them over the course of the guide. But for now, let's focus on four of them.

Four suffixes

Whenever we use a suffix, we must keep a few things in mind:

  • what the suffix means. Suffixes take basic ideas like "go" and create more complex ones, like "the one who goes" or "the means of going."
  • what the suffix does. Suffixes often cause some sort of vowel change, especially when they're attached directly to a root. But they may cause other sorts of changes too.
  • how the suffix behaves. Suffixes usually create words of a particular gender.

As you continue, remember that these ideas are guidelines, not hard rules. Sanskrit is a human language, and like all human languages, Sanskrit can behave in unpredictable ways.

{{ '-a'|i }}

The {{ '-a'|i }} suffix is one of the most complicated suffixes in Sanskrit. It has several different meanings and can cause all sorts of sound changes. But this suffix also gives us some very common words, like {{ 'yoga'|i }} and {{ 'veda'|i }}. It is an important suffix and one worth knowing well.

Usually, {{ '-a'|i }} describes a state of being, and it strengthens the root vowel until it is part of a heavy syllable. Also, it changes {{ 'cavarga'|i }} sounds to {{ 'kavarga'|i }} sounds. The results are almost always masculine:

    {{ ex('zuc -> zoka', 'grieve -> grief') }} {{ ex('muh -> moha', 'be confused or deluded -> confusion, delusion') }} {{ ex('sRj -> sarga', 'emit, produce -> emission, production, creation') }} {{ ex('labh -> lAbha', 'obtain -> obtaining, acquisition, gain') }}

In Sanskrit grammar, this "state of being" is called {{ 'bhAva'|i }}. So we could say that {{ '-a'|i }} is added in the sense of {{ 'bhAva'|i }}. Incidentally, the word {{ 'bhAva'|i }} itself comes from the {{ '-a'|i }} suffix:

    {{ ex('bhU -> bhAva', 'be -> state of being') }}

Now for other examples. {{ '-a'|i }} often creates words with predictable meanings, but sometimes these meanings shift in different directions:

    {{ ex('man -> mAna', 'think -> thought, idea; pride, arrogance') }} {{ ex('dRz -> darza', 'see -> sight; appearance') }}

{{ '-a'|i }} can also create more concrete words, as well:

    {{ ex('cur -> cora', 'steal -> thief') }} {{ ex('visRj -> visarga', "set free, emit -> \"sending forth,\" the {{ 'visarga'|i }}"|render) }}

And it can also cause other sorts of vowel changes.

    {{ ex('kR -> kara', 'do -> means of doing; a hand') }} {{ ex('bhU -> bhava', 'be -> everything that is; worldly existence') }}

{{ '-a'|i }} is a rich suffix, but we've explored enough of it for now. Let's look at the other three suffixes instead. These suffixes are much simpler than {{ '-a'|i }}.

{{ '-aka'|i }}

Most often, {{ '-aka'|i }} creates masculine nouns. Like {{ '-a'|i }}, {{ '-aka'|i }} usually strengthens the root vowel until it is in a heavy syllable. {{ '-aka'|i }} attaches to some root "X" to create a word that means "one who does X":

    {{ ex('kR -> kAraka', 'do -> one who does') }} {{ ex('pac -> pAcaka', 'cook -> one who cooks; cooker, cook') }} {{ ex('nI -> nAyaka', 'lead -> one who leads; leader, guide, hero') }}

{{ '-ana'|i }}

Most often, {{ '-ana'|i }} creates neuter nouns and strengthens the root vowel just once. Usually, {{ '-ana'|i }} creates words that mean "the act of doing X" or "a case of doing X":

    {{ ex('jan -> janana', 'be born -> (the act of) producing something; birth') }} {{ ex('ji -> jayana', 'conquer -> (the act of) conquering') }} {{ ex('jJA -> jJAna', 'know -> (the act of) knowing') }} {{ ex('dRz -> darzana', 'see -> (the act of) seeing') }} {{ ex('nI -> nayana', 'lead -> (the act of) leading') }} {{ ex('muh -> mohana', 'stand -> (a case of) being confused; confusion, delusion') }} {{ ex('yudh -> yodhana', 'fight -> (the act of) fighting; war') }} {{ ex('zru -> zravaNa', 'hear -> (the act of) hearing') }} {{ ex('sthA -> sthAna', 'stand -> (the act of) standing firmly') }}

But many other sorts of meanings exist too:

    {{ ex('kR -> karaNa', 'do -> a means of doing something; an instrument') }} {{ ex('car -> caraNa', 'walk -> a means of walking; foot') }} {{ ex('jJA -> jJAna', 'know -> what is known; knowledge') }} {{ ex('sthA -> sthAna', 'stand -> a place where one stands; position, state, condition') }}

{{ '-ana'|i }} creates other sorts of words, but the ones above are the most important to remember. This suffix forms many words in the Hindu lexicon, like {{ 'darzana'|i }}, {{ 'upanayana'|i }}, {{ 'agnicayana'|i }}, {{ 'rAmAyaNa'|i }}, and many others.

{{ '-tra'|i }}

Most often, {{ '-tra'|i }} creates neuter nouns and strengthens the root vowel just once. Usually, {{ '-tra'|i }} creates words that mean "a means of doing X". Like {{ '-ana'|i }}, {{ '-tra'|i }} is used to create many important words:

    {{ ex('nI -> netra', 'lead -> a means of leading; an eye') }} {{ ex('zru -> zrotra', 'hear -> a means of hearing; an ear') }} {{ ex('man -> mantra', 'think -> a means of thinking; a mantra') }}

Other suffixes

We have seen four root suffixes so far. But Sanskrit has dozens more. Some of them have no clear meaning:

    {{ ex('dhR (dharati) -> dharma', 'bear, carry, keep firm -> that which is firm; dharma') }}

But others are subtle and extremely powerful. One of these powerful suffixes is {{ '-ta'|i }}, which is used to create "Sanskrit" ({{ 'saMskRta'|i }}) itself. But to prepare for {{ '-ta'|i }} and some of these other suffixes, we have to learn a little more about Sanskrit verbs. We'll do so over the next few lessons.
