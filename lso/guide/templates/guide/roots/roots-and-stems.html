Learning Sanskrit is like building a massive house. Our materials are the Sanskrit sounds, which we combine and blend to create stems, endings, words, and sentences. Our techniques are the rules of grammar, which have stayed simple so far. And our approach is still a practical one: build a basic shelter before expanding further.

Now we have built our foundation, and there are many ways to expand. But let's return to the humble verb, where we started our journey into Sanskrit:

    {{ ex('khAdanti', 'They eat.') }}

The verb is {{ 'khAdanti'|i }} and the stem is {{ 'khAda'|i }}. But there is more to see than just {{ 'khAda'|i }}:

    {{ ex('mRgAH khAdanti', 'The deer eat.') }} {{ ex('naraH khAdiSyati', 'The man will eat.') }} {{ ex('puruSaH khAdyate', 'The man is eaten.') }} {{ ex('pazyAmi mRgasya khAdakam', 'I see the eater of the deer.') }} {{ ex('siMhaH khAditaH', 'The lion has been eaten.') }} {{ ex('gajaH khAditvA gacchati', 'The elephant goes after eating.') }}

In all of these words we see something more subtle than the verb stem. It is a basic core that seems to endure. This is the root:

    {{ ex('khAd -> khAda -> khAdati', 'He eats.') }}

Creating roots

Just as a single tree trunk produces many branches and flowers, a single root produces many stems and words. But just as the base of the tree remains underground and hidden from view, so too is it difficult to tell what a root really is. In the forms above, the root is obviously {{ 'khAd'|i }}. But consider the forms below:

    {{ ex('vIrA jayanti', 'The heroes conquer.') }} {{ ex('putrau jeSyataH', 'The two sons will conquer.') }} {{ ex('grAmo jIyate', 'The village is conquered.') }} {{ ex('pazyAmi jayakam', 'I see the victor.') }} {{ ex('loko jitaH', 'The world has been conquered.') }} {{ ex('naro jitvA gacchati', 'The man goes after conquering.') }}

It is easy to think of a few possible roots. But consider the difficulties involved:

  1. We must choose a single root that can produce {{ 'jay'|i }}, {{ 'je'|i }}, {{ 'jI'|i }}, and {{ 'ji'|i }}.
  2. We should make rules to explain how our root transforms into {{ 'jay'|i }}, {{ 'je'|i }}, {{ 'jI'|i }}, and {{ 'ji'|i }}.
  3. And these rules must be as consistent as possible across all roots and all of the stems that those roots produce.

This is an enormous task. But fortunately, the grammarian Panini has done the work for us. Panini defined a full list of verb roots, which can let us know how Panini chose roots. And many of the rules of the Ashtadhyayi are concerned with transforming these roots into the words we really see.

Panini did not describe how he chose his roots. But by studying his list, we can create some simple guidelines. Here is one of them:

Roots should use simple vowels. We can then form stems by strengthening the vowel.

Here, we strengthen a vowel by adding an {{ 'a'|i }} vowel to the front of it.

With this guideline in mind, we can find a root for {{ 'jayati'|i }}:

    {{ ex('ji -> jayati', 'He conquers.') }}

Now consider the words below:

    {{ ex('azvA nayanti', 'The horses lead.') }} {{ ex('naro neSyati', 'The man will lead.') }} {{ ex('sundaro nIyate', 'The beautiful one is led.') }} {{ ex('rakSasi nAyakam', 'You protect the leader.') }} {{ ex('zvetagajo nItaH', 'The white elephant has been led.') }} {{ ex('naro nItvA mRgaM labhate', 'The man obtains a deer after leading.') }}

Here is another basic guideline:

Roots should contain as much useful information as possible.

With this guideline in mind, we can find a root for {{ 'nayati'|i }}:

    {{ ex('nI -> nayati', 'He leads.') }}

Finally, consider the words below:

    {{ ex('gRhebhyo gacchAmaH', 'We go from the houses.') }} {{ ex('naro gamiSyati', 'The man will go.') }} {{ ex('devo gamyate', 'The god is gone to.') }} {{ ex('pazyAvaH gamakam', 'The two of us see the indicator.') }} {{ ex('gRho gataH', 'The house has been gone to.') }} {{ ex('narA gatvA pacanti', 'The men cook after going.') }}

Here is one final guideline:

When there's no clear or simple pattern, roots should be something common.

With this in mind, we can find a root for {{ 'gacchati'|i }}:

    {{ ex('gam -> gacchati', 'He goes.') }}

Irregular roots

Sometimes, the words involved are too difficult to explain:

    {{ ex('tiSThanti vinA naraiH', 'They stand without the men.') }} {{ ex('naraH sthAsyati', 'The man will stand.') }} {{ ex('naraH sthIyate', 'The man is stationed.') }} {{ ex('pazyAmi sthAnam', 'I see the place.') }} {{ ex('naraH sthitaH', 'The man is standing.') }} {{ ex('naro sthitvA zocati', 'The man grieves after standing.') }}

Forms like these exist because Sanskrit is an old language and we do not fully understand its early history. Panini coped with these forms by adding extra rules and exceptions. He defined the root of {{ 'tiSThati'|i }} like this:

    {{ ex('sthA -> tiSThati', 'stand -> He stands.') }}

But it would take too long to explain why, so let's leave the discussion here for now.

Some changes can also seem totally bizarre:

    {{ ex('kRSNAH pazyanti', 'The black ones see.') }} {{ ex('naro drakSyati', 'The man will see.') }} {{ ex('naro dRzyate', 'The man is seen.') }} {{ ex('corayAmi darzakAya', 'I steal for the one who sees.') }} {{ ex('naro dRSTaH zareSu', 'The man has been seen among the arrows.') }} {{ ex('devAnAM gajaM dRSTvA manye', 'I think after seeing the elephant of the gods.') }}

The roots here are {{ 'paz'|i }} and {{ 'dRz'|i }}. This is a strange case where two roots have started to complement each other. It's like two trees fusing into one. The convention is to refer to these two with just {{ 'dRz'|i }}:

    {{ ex('dRz -> pazyati', 'see -> He sees.') }}

Panini's system has some flaws, and it is not the only system available. But regardless of its imperfections, it has been extremely successful for thousands of years. All Sanskrit texts use it, and this guide will do the same.
