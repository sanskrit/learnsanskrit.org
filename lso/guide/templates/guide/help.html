Using the Sanskrit grammar guide

This page is here to help you have the best experience possible when using the Sanskrit grammar guide. It also has information about using the guide offline. If you've read through this page and have some questions, let us know what we can do to help.

  • Programs and software
  • Downloading the guide

Programs and software

For the best possible experience, please use a modern web browser. Other than that, there's little to do.

If you're using a very old operating system (before Mac OS 10.3 or Windows 2000), then you should also make sure that you have Devanagari installed. If this:

{{ 'kazcitkAntAvirahaguruNA svAdhikArapramattaH
zApenAstaMgamitamahimA varSabhogyeNa bhartuH |'|d }}

looks like rubbish, then you probably don't have Devanagari installed. For help, you can consult this FAQ page from Wikipedia. Once you have Devanagari installed, you can use the site normally.

Downloading the guide

By popular request, you can now download the grammar guide for offline use. These files can be used with printers and e-readers, and they can be used in any situation where an ordinary website is too difficult to handle. Of course, these files are missing some of the things that make web pages so special: sound and video, interactive content, color, and so on.

If you want to dowload the guide, you have two options available:

Option 1: PDFs from learnsanskrit.org

The easier option is to use PDFs that have been created by learnsanskrit.org. These PDFs render Sanskrit in Devanagari and IAST. They are updated regularly, but they are not always as up-to-date as the online lessons.

  • Grammar guide
    The complete grammar guide. Devanagari and IAST.

  • Sounds
    All lessons from the "Sounds" unit of the guide. Devanagari and IAST.

  • Basics
    All lessons from the "Basics" unit of the guide. Devanagari and IAST.

Option 2: Make your own PDFs

The more powerful option is to create your own PDFs. You can use whatever scripts you like, and your PDFs are guaranteed to be as up-to-date as possible.

First, you should set your settings and make sure the guide looks as you expect. If you select File -> Print in your browser, the current page will format itself to be printer-friendly.
