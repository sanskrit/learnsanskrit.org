By using nouns, we can describe the things involved with an action. We have already studied case 1 and case 2:

    {{ iex('grAmaM rakSanti vIrAH', 'The heroes protect the village.') }} {{ iex('gRhaM pazyAmi', 'I see the house.') }} {{ ihex('siMhAs tiSThanti', 'The lions stand.') }}

Case 1 and case 2 define very basic relationships. But the remaining cases are more interesting.

Case 3

Case 3 has two important senses. More commonly, case 3 defines something used in the action: {%- filter foot(notes) %} Ashtadhyayi 1.4.42 : {{ 'sAdhakatamaM karaNam'|i }} "What is most useful is called {{ 'karaNa'|i }}," or "instrument." This is one of the roles that case 3 expresses. {% endfilter -%}

    {{ iex('gajena gacchAmi grAmam', 'I go by means of the elephant to the village.') }} {{ iex('gacchatho grAmAn rathAbhyAm ', 'The two of you go by means of the two chariots to the villages.') }} {{ iex('azvaM labhante naraiH', 'They obtain a horse by means of the men.') }}

Less commonly, case 3 defines a second performer in the action:

    {{ iex('azvena tiSThAvaH', 'The two of us stand with the horse.') }} {{ ihex('vIrAbhyAM jayasi', 'You conquer with the two heroes.') }} {{ ihex('manyadhve vIraiH', 'You all think with the heroes.') }}

This second sense can be reinforced by using {{ 'saha'|i('dfn') }}:

    {{ iex('gajena saha gacchAmi grAmAn', 'I go with the elephant to the villages.') }} {{ ihex('sahAzvAbhyAM tiSThanti', 'They stand with the two horses.') }}

And it can be reversed by {{ 'vinA'|i('dfn') }}:

    {{ iex('caranti vinA vIraiH', 'They walk without the heroes.') }} {{ ihex('gajena vinA gacchAmi', 'I go without the elephant.') }}

{{ 'saha'|i }} and {{ 'vinA'|i }} should be very close to the word in case 3. Usually, they appear right after. These words are also uninflected words: they never take endings of any kind.

Stem-ending blending: {{ 'n'|i }} to {{ 'N'|i }}

We have already seen many different kinds of blending. All of the examples we've seen have one thing in common: blending occurred between words:

    {{ ihex('vIraH rathau labhate -> vIro rathau labhate', 'The hero obtains the two chariots.') }}

But blending can also occur within a word. Consider the words below:

    {{ iex('narena', 'with the man') }} {{ iex('vIrena', 'with the hero') }}

Try repeating the first word ten or twenty times. Eventually it will become tiresome to pronounce the retroflex {{ 'r'|i }} right before shifting to the non-retroflex {{ 'n'|i }}. It is much easier to blend the two sounds by using the same point of sound for both:

    {{ ihex('narena -> nareNa', 'with the man') }} {{ ihex('vIrena -> vIreNa', 'with the hero') }}

With this in mind, we can understand the sentences below:

    {{ ihex('saha nareNAzvAn rakSAmi', 'I protect the horses with the man.') }} {{ ihex('vIreNa vinA gajaM labhase', 'You obtain the elephant without the hero.') }}

Case 4

Case 4 usually defines the person meant to benefit from the action: {%- filter foot(notes) %} Ashtadhyayi 1.4.32 : {{ 'karmaNA yam abhipraiti sa saMpradAnam'|i }} "What the agent has in mind when performing the action is called {{ 'saMpradAna'|i }}," or "presentation." This is one of the roles that case 4 expresses. {% endfilter -%}

    {{ iex('gajAya manye', 'I think for the man.') }} {{ iex('naraH pacati vIrAbhyAm ', 'The man cooks for the two heroes.') }} {{ iex('siMhaM jayAvaH gRhebhyaH', 'The two of us conquer the lion for the villages.') }}

Compared to the other noun cases, case 4 is uncommon. But you should still know it well!

    {{ ihex('pacati grAmAya gajam', 'He cooks the elephant for the village.') }} {{ ihex('vIrAbhyAM grAmaM gacchathaH', 'The two of you go to the village for the two heroes.') }} {{ ihex('narebhyo vIrAH siMhaM jayanti', 'The heroes conquer the lion for the men.') }}

Ambiguities

Like most languages, Sanskrit can be ambiguous. We have seen ambiguous behavior before:

    {{ iex('gajau labhete', '(a) The two of them obtain the two elephants.
    (b) The two elephants obtain.') }}

Words like {{ 'gajAbhyAm'|i }} and {{ 'narAbhyAm'|i }} are ambiguous, too:

    {{ iex('narAbhyAM pacanti', '(a) They cook for the two men.
    (b) They cook with the two men.') }}

Usually, common sense is enough to tell what a word should mean:

    {{ ihex('gajAbhyAM pacanti', 'They cook for the two elephants.') }}
