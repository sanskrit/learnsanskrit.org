This unit gives us a basic foundation for the Sanskrit language. We learn the fundamental concepts that make Sanskrit what it is and study some interesting examples of how these concepts are used. We also learn some basic words and start writing simple sentences.

It is essential to understand these concepts perfectly. So if you have any doubts about what something means, or if you think you've found an error, please let us know what we can do to help.
