Using the Sanskrit we know already, we can write a variety of sentences:

    {{ ihex('pazyati', 'He sees.') }} {{ ihex('labhe', 'I obtain.') }} {{ ihex('pacase', 'You cook.') }}

By using nouns, we can describe the things involved with this action. We just learned about case 1, which generally describes who performs the action:

    {{ ihex('gajaz carati', 'The elephant walks.') }} {{ ihex('narau labhete', 'The two men obtain.') }} {{ ihex('azvA gacchanti', 'The horses go.') }}

Meanwhile, case 2 usually defines the "object" of the action:

    {{ iex('gajaM labhadhve', 'You all obtain an elephant.', aside="({{'m'|i }} blends with {{ 'p'|i }} to form {{ 'M'|i }}.)") }} {{ iex('gajau pazyasi', 'You see two elephants.', aside="({{'gajau'|i }} is the same in case 1 and case 2!)") }} {{ iex('gajAn pazyathaH', 'The two of you see elephants.') }}

With verbs that imply movement (like "go," "come," "walk," and so on), case 2 also defines the destination:

    {{ iex('naraM gacchatha', 'You all go to the man.') }} {{ ihex('narau gacchAmi', 'I go to the two men.') }} {{ ihex('narAn gacchAvaH', 'The two of us go to the men.') }}

Ambiguity

Above, we saw that {{ 'gajau'|i }} appears in case 1 and case 2. This can create ambiguities in Sanskrit, where multiple meanings are possible:

    {{ iex('gajau labhete', '(a) The two of them obtain the two elephants.
    (b) The two elephants obtain.') }}

These ambiguities disappear if we know more about this sentence's context. If we do not have enough context, then there is nothing we can do. But fortunately, these sorts of ambiguities are also quite rare. They go away if the verb changes even slightly:

    {{ ihex('gajau labhethe', 'The two of you obtain the two elephants.') }} {{ ihex('gajau labhante', 'They obtain the two elephants.') }}
