Generally, every Sanskrit sentence is just some action. The simplest sentences are just actions and nothing else:

    {{ ihex('pazyati', 'He sees.')}} {{ ihex('labhethe', 'The two of you obtain.')}} {{ ihex('manyAmahe', 'We all think.')}} {{ ihex('tiSThasi', 'You stand.')}}

Unfortunately, these sentences tell us very little. If we see just {{ 'pazyati'|i }}, how can we tell who sees, or what is seen? And how and where does this "seeing" happen?

To fill these gaps, we use nouns. Nouns describe ideas as simple as elephants and fruits, and as complex as places and concepts. Just like verbs, nouns are inflected:

    {{ iex('gajaH pazyati', 'The elephant sees.') }} {{ iex('gajau pazyataH', 'The two elephants see.') }} {{ iex('gajAH pazyanti', 'The elephants see.') }}

And just like verbs, nouns have a stem. Here, the stem is {{ 'gaja'|i }}. Just as many flowers bloom from the same plant stem, many nouns are formed from the same noun stem. And as with verbs, we add an ending to a noun to make a complete word:

    {{ ihex('gajaH pazyati', 'The elephant sees.') }} {{ ihex('gajau pazyataH', 'The two elephants see.') }} {{ ihex('gajAH pazyanti', 'The elephants see.') }} {{ iex('naraH pacate', 'The man cooks for himself.') }} {{ ihex('narau pacete', 'The two men cook for themselves.') }} {{ ihex('narAH pacante', 'The men cook for themselves.') }} {{ iex('azvas tiSThati (azvaH tiSThati)', 'The horse stands.') }} {{ ihex('azvau tiSThataH', 'The two horses stand.') }} {{ ihex('azvAs tiSThanti (azvAH tiSThanti)', 'The horses stand.') }}

Note the blending that occurs in {{ 'azvas tiSThati'|i }} and {{ 'azvAs tiSThati'|i }}. Now that our sentences have multiple words, we have to be mindful of how the sounds in the two words affect each other.

Noun roles

In each example above, the noun defines what performs the verb action. But nouns can define other parts of the action, too. With {{ 'pazyati'|i }}, for example, we can define what is seen:

    {{ iex('gajaH pazyati naram', 'The elephant sees the man.') }}

Or where the elephant sees:

    {{ iex('grAme gajaH pazyati', 'The elephant sees in the village.') }}

Or what the elephant sees with:

    {{ iex('nayanena gajaH pazyati', 'The elephant sees with his eye.') }}

Here, the role of the noun changes when we change the noun ending. So in addition to showing the number of items involved, the noun ending shows the noun's role. Generally, these roles are called cases. Still, roles and cases are not quite the same thing.

Case 1

Case 1 usually defines what performs the action:

    {{ iex('gajaH pazyati', 'The elephant sees.') }} {{ iex('gajau pazyataH', 'The two elephants see.') }} {{ iex('gajAH pazyanti', 'The elephants see.') }}

{{ 'pazyati'|i }} implies that only one thing sees. {{ 'gajaH'|i }} implies that only one elephant performs the action. Each word implies the same number of things (one), so they are used together. A verb and a word in case 1 must always imply the same number of things.

Blended sounds

Recall that {{ 'visarga'|i }} blends with the sounds that follow it:

    {{ iex('azvazcarati', 'The horse walks.') }} {{ ihex('azvau carataH', 'The two horses walk.') }} {{ ihex('azvAzcaranti', 'The horses walk.') }} {{ iex('gajastiSThati', 'The elephant stands.') }} {{ ihex('gajau tiSThataH', 'The two elephants stand.') }} {{ ihex('gajAstiSThanti', 'The elephants stand.') }}

In these examples, the consonant that comes after the {{ 'visarga'|i }} is not voiced. What if the consonant is voiced instead?

    {{ iex('naraH labhate', 'The man obtains.') }} {{ iex('narAH labhante', 'The men obtain.') }}

Try repeating these entences ten or twenty times. Eventually it will become tiresome to transition between the unvoiced {{ 'visarga'|i }} in {{ 'naraH'|i }} and {{ 'narAH'|i }} and the voiced {{ 'l'|i }} in {{ 'labhate'|i }} and {{ 'labhante'|i }}. It is much easier to blend the two words.

For the first sentence, this blending is against all of our intuitions. It is the sort of change we must just accept.

    {{ ihex('naraH labhate -> naro labhate', 'The man obtains.') }}

The {{ '-aH'|i }} ending became {{ 'o'|i }}. This happens in front of voiced consonants of any kind.

For the second sentence, the blending is much more reasonable:

    {{ ihex('narAH labhante -> narA labhante', 'The men obtain.') }}

Here the troublesome {{ 'visarga'|i }} is simply removed. Sometimes the best solution to a problem is the easiest one.

With these changes in mind, we can now write many more sentences:

    {{ ihex('naro manyate', 'The man thinks.') }} {{ ihex('narau pazyataH', 'The two men see.') }} {{ ihex('narA manyante', 'The men think.') }} {{ ihex('gajo labhate', 'The elephant obtains.') }} {{ ihex('gajau labhete', 'The two elephants obtain.') }} {{ ihex('gajA labhante', 'The elephants obtain.') }} {{ ihex('azvo gacchati', 'The horse goes.') }} {{ ihex('azvau tiSThataH', 'The two horses stand.') }} {{ ihex('azvA gacchanti', 'The horses go.') }}
