Many of the examples below will depend on your computer's support for Devanagari. Ordinary Devanagari fonts usually do not support more complicated clusters, perhaps with good reason.

Sanskrit sounds are grouped together into clusters:

    {{ iex('mAmakAH -> mA ma kAH') }}

These clusters can have more than one consonant. And in Devanagari, these consonants are mashed together:

    {{ iex('kRSNa -> kR SNa')}}

Unfortunately, consonant clusters can be even more difficult than in this example. Some clusters are so densely combined that the original letters are unrecognizable. And there are simply too many to memorize:

Fortunately, most consonant clusters fall into simple patterns. By understanding these patterns, we can learn these clusters much more quickly.

This lesson will cover the basic patterns behind consonant clusters. By learning these patterns, you should be able to figure out most all of the clusters you will see in Sanskrit.

Basic patterns

Horizontal

Most Devanagari letters have a vertical bar that goes down the side or middle of the letter.

    {{ iex('kha, ga, gha, ca, ja, jha, Ja, Na, ta, tha, dha, na, pa, ba, bha, ma, ya, la, va, za, Sa, sa') }} {{ iex('ka, pha') }}

In the simplest sort of conjunct, this stick and everything to the right of it is removed. The letters are then attached horizontally.

    {{ iex('s ya -> sya') }} {{ iex('g dha -> gdha') }} {{ iex('t m ya -> tmya') }}

Vertical

In some characters — especially where this vertical "stick" is missing — the two letters attach vertically:

    {{ iex('S Ta -> STa') }} {{ iex('p ta -> pta') }} {{ iex('G ga -> Gga') }} {{ iex('d bha -> dbha') }}

Some of these combinations are rather tricky:

    {{ iex('d ya -> dya') }} {{ iex('d ma -> dma') }} {{ iex('h ya -> hya') }} {{ iex('h ma -> hma') }}

Horizontal and vertical

In longer clusters, letters may come together both horizontally and vertically:

    {{ ex('G kh ya -> Gkhya') }}

Irregular combinations

Some clusters are difficult enough that they cannot be read correctly on the first try:

{{ 'kSa'|d }} and {{ 'jJa'|d }}

These two clusters are very common. Note that the individual letters are unrecognizable:

    {{ row('kSa jJa') }}

Consonant + {{ 'ra'|d }}

When {{ 'ra'|i }} appear before a consonant, the combination is simple:

    {{ iex('r + pa -> rpa') }}

When {{ 'ra'|d }} appears after a consonant, the combination is slightly trickier. {{ 'ra'|d }} is attached as a tick-mark on the "stick" part of the cluster before it:

    {{ iex('p + ra -> pra') }} {{ iex('m + ra -> mra') }}

If no stick is available, {{ 'ra'|d }} is written slightly differently:

    {{ iex('ST + ra -> STra') }}

{{ 'tta'|d }}, {{ 'kta'|d }}, {{ 'tra'|d }}, and {{ 'ktra'|d }}

These conjuncts just barely show the original letters.

    {{ row('tta kta tra ktra') }}

{{ 'za'|d }} + various
