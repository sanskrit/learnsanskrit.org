When used to write Vedic texts, Devanagari can be very complex. This lesson introduces a few simple symbols that can help you passably read Vedic Devanagari.

Between vowels, {{ 'Da'|d }} and {{ 'Dha'|d }} usually become {{ 'La'|d }} and {{ 'Lha'|d }}:

    {{ row('La Lha') }}

Accent

Unlike "ordinary" Sanskrit, Vedic Sanskrit has an accent system, where some vowels are spoken with a higher pitch than others. We denote these accents with some special marks:

  • क
  • क॒
  • क॑

Each of these symbols denotes a different accent on the letter {{ 'ka'|sa1 }}. Usually, the first example is the {{ 'udAtta'|i('dfn') }}, the second is the {{ 'anudAtta'|i('dfn') }}, and the third is the {{ 'svarita'|i('dfn') }}.
